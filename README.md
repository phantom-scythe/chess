# [FEN-String-Simulator](https://github.com/phantom-scythe/FEN-String-Simulator)
Enter your FEN String into the code and the FEN String is shown immediately onto the board.

# Community
Feel free to provide anything that you think might make the game better. This was originally supposed to be a game where you were
supposed to face off against a custom built bot but, I have hit a wall and the complexity has overwhelmed me. I would love it, if 
anyone tried to help, no matter how small it is.
